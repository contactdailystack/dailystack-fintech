## DailyStack: Design Concept (v6.8 Unified Blueprint with Wallet Integration)
"Premium Fintech meets AI-Native Lifestyle & Relationships Experience"

---

### 1. Design Philosophy (ปรัชญาการออกแบบ)

การออกแบบของ **DailyStack** ยึดมั่นบน 4 แกนหลักที่ขับเคลื่อนตั้งแต่ก้าวแรกของผู้ใช้ไปจนถึงการใช้งานร่วมกันในระยะยาว:

1.  **Dark-First Cinematic Interface** — พื้นหลังสีเข้มลึก (Deep Void `#0d1117` และ `#131313`) เพื่อลดความเหนื่อยล้าสายตา ขับเน้นองค์ประกอบภาพและสร้างมิติความลึก (Depth) ให้ความรู้สึกระดับสากลและหรูหราแบบ Premium
2.  **Electric Neon Accent & Feature Palettes** — การใช้สีอย่างจงใจเพื่อนำทางความสนใจ (Guided Attention) โดยใช้สีเขียวนีออน (`#C0F500`) สำหรับปุ่มปฏิบัติการหลัก (CTA) และใช้สีเฉพาะประจำตัวมิติชีวิต (Feature Palettes) เพื่อจัดระเบียบความคิดและอารมณ์ของผู้ใช้งาน
3.  **Gamified Fintech & Status Indicators** — เปลี่ยนเป้าหมายชีวิตและเรื่องความสัมพันธ์ให้รู้สึกจับต้องได้ มีมูลค่า และสนุกสนาน ผ่านการจำลองบัตรเครดิตพรีเมียม (Virtual Platinum Card) และระบบคะแนนพลังงาน (Energy Points Ledger) ที่ลื่นไหล
4.  **Premium Wealth Companion & Asset Security** — ระบบจัดการทรัพย์สินและการเงินส่วนบุคคล (Wallet Feature) ที่ผสานการจดบันทึกรายรับ-รายจ่ายระดับบัญชีคู่ (Double-entry Bookkeeping) เข้ากับดีไซน์เรียบหรู ปลอดภัย และแสดงผลสถิติที่สวยงาม คมชัด ตามแบบฉบับแอประดับโลกอย่าง Rocket Money

### 2. Color Strategy (กลยุทธ์การใช้สี)

การใช้สีจัดแบ่งตามระบบ Layer ลึกตื้นและการทำกิจกรรมต่าง ๆ เพื่อสร้าง Visual Hierarchy ที่ชัดเจนโดยไม่ต้องใช้เส้นขอบหรือเงาที่หนาจนเกินไป:

#### Core System Layers
```plaintext
Background Layer (Deepest Void):  #0d1117 (Auth & Main Pages) / #131313 (Onboarding & Swipe)
Surface Layer (Cards & Panels):    #161b22 (Deep Cards) / #242424 (Surface Cards)
Interactive Layer (Inputs & Pills): #21262d (Input fields & Interactive pills)
Border Layer (Subtle Separation):   #30363d (Subtle borders)
Primary Accent (CTA & Success):    #C0F500 (Electric Lime / Energy Actions)
Text Primary (High Contrast):      #F5F5F7 / #F0F6FC
Text Secondary (Supporting):       #8B949E / #9CA3AF
Error (Clear Warnings):            #F85149 / #EF4444
```

#### Feature Palettes & Wallet Colors
เพื่อจำแนกและกระตุ้นพฤติกรรมในแต่ละมิติ (Feature Feature Identity) ให้มองเห็นได้ทันทีทางจิตวิทยา:
*   **Work & Creation Stack**: นีออนมะนาว (`#C0F500`) — แทนคุณค่า ความมีประสิทธิผล และการลงมือสร้าง
*   **Learning & Growth Stack**: มินต์กรีน (`#56BE89`) — แทนการเจริญเติบโต ความรู้ และความงอกงาม
*   **Relationships & Dating Stack**: ชมพูปะการัง (`#FF6B9D` / `#FF6B81`) — แทนความรู้สึกอบอุ่น พลังใจ และการเชื่อมโยง
*   **Passions & Legacy Stack**: อเมทิสต์ม่วง (`#9B5DE5`) — แทนพลังแห่งจินตนาการ สิ่งสร้างสรรค์ และความฝัน
*   **Well-being Stack**: เหลืองอำพัน (`#FFB800`) — แทนพลังงาน ความสดชื่น และการพักฟื้นสุขภาพ
*   **Wallet (Expense & Budget) Feature**: แดงปะการังเข้ม (`#D6453E`) และแอปริคอตซอฟต์ (`#FBF5F1` สำหรับไลต์โหมด) — แทนความมั่นคงทางการเงิน ความรอบคอบ และพลังแห่งสินทรัพย์สุทธิ
*   **Ultra Match & Rewards**: ทองสเปเชียล (`#D9FD82` / `#FFD700`) — บ่งบอกถึงศักยภาพความเข้ากันได้สูงสุดและการได้รับรางวัลพิเศษ

### 3. Typography System (ระบบอักษร)

| Role (หน้าที่) | Font (ฟอนต์) | Usage (การใช้งาน) | Psychology (จิตวิทยา) |
| :--- | :--- | :--- | :--- |
| **Headings** | Space Grotesk / Kanit | ตัวหนาพิเศษ, เรขาคณิต, ชื่อแบรนด์และหัวข้อหลัก | ทันสมัย แข็งแกร่งแบบวิศวกรรมที่เข้าถึงง่าย |
| **Body** | Space Grotesk / Sans | อ่านง่าย สบายตา สม่ำเสมอทุกหน้าจอ | มีสไตล์ มีความแม่นยำสูง แต่ไม่แข็งทื่อ |
| **Technical** | JetBrains Mono | ความแรงรหัสผ่าน, ตัวเลขนับถอยหลัง Lockout, ยอดเงินและตารางบัญชีคู่ (Wallet ledger) | แสดงถึงความปลอดภัยขั้น Fintech และความแม่นยำสูงสุดของตัวเลขทางการเงิน |

### 4. UX Patterns & Customer Journey Alignment

การออกแบบ Interaction ตลอดการเดินทางของผู้ใช้งาน (Customer Journey) ถูกจำลองขึ้นมาเพื่อตอบสนองความรู้สึกทางจิตวิทยาโดยละเอียด:

#### 1. Progressive Disclosure (การเปิดเผยทีละขั้นเพื่อลดความอึดอัด)
*   **Authentication**: แบ่งเป็นกระบวนการสั้น ๆ `Email` $\rightarrow$ `Password` $\rightarrow$ `Confirm & Go` เพื่อลด Cognitive Load
*   **Onboarding**: แบ่งหน้าจอเก็บข้อมูลออกเป็น 5 มิติการ sync อย่างเพลิดเพลิน แทนที่จะให้กรอกฟอร์มขนาดยาวในครั้งเดียว (Basic $\rightarrow$ Photos $\rightarrow$ Lifestyle $\rightarrow$ Interests $\rightarrow$ Done)

#### 2. Habit-to-Match Visual Loop (เชื่อมการกระทำกับผลลัพธ์)
*   **Progressive Rings**: วงกลมบน Home Dashboard จะปิดรอบตัวเองเมื่อผู้ใช้สะสมเป้าหมายสำเร็จในแต่ละวัน สะท้อนออกมาเป็นตัวเลข Energy Points ที่เพิ่มขึ้นในระบบกระเป๋าเงินเสมือน
*   **AI Match Radar**: แสดงแถบพลังความเข้ากันได้ของเวลานอนและสไตล์การทำงานจำลองระหว่างผู้ใช้ทันที เพื่อเน้นย้ำถึงแก่นแท้ของแอปพลิเคชัน

#### 3. Wallet Specific Interactions (ระบบเงินและสินทรัพย์อัจฉริยะ)
*   **Masking Mode (โหมดซ่อนยอดเงิน)**: ผู้ใช้สามารถกดปุ่มรูปดวงตาแบบ Cyberpunk เพื่อเบลอยอดเงินรวมและทรัพย์สินทั้งหมดด้วยเทคนิคกระจกฝ้า (Glassmorphism Blur) ป้องกันการสอดแนมเมื่อเปิดใช้งานในที่สาธารณะ
*   **Smart OCR Entry feedback**: แสดงแอนิเมชันสแกนเลเซอร์สีแดงปะการังบนรูปภาพใบเสร็จ/สลิปขณะทำการดึงข้อมูลประเภทบัญชี วันที่ และจำนวนเงินโดยอัตโนมัติ
*   **Predictive Alert Warning**: เมื่อระดับค่าใช้จ่ายสูงเกิน 80% ของงบประมาณ แดชบอร์ดจะปรับเปลี่ยนการ์ดรายงานเป้าหมายเป็นสีเหลืองอำพันพร้อมการแสดงผลวิเคราะห์แนวโน้มที่คาดว่าจะเกินงบ (Trend Analysis indicator)

#### 4. Post-Match Couple Life Expansion
*   **Couple Transition & Collaborative Budgeting**: ระบบเปลี่ยนผ่าน UI จาก "การค้นหาค้นเดทเดี่ยว" ไปสู่ "การสร้างสเปซร่วมกันของคู่รัก" (Shared Dashboard & Goal tracking) เพื่อแก้ปัญหาวงจรอัตราการละทิ้งแอปหาคู่ทั่วไป และสร้างพื้นที่การวางแผนงบประมาณร่วมกันในอนาคต (Shared Wallets)

### 5. Motion Design (การเคลื่อนไหวที่มีความหมาย)

แอนิเมชันทุกจุดทำงานประสานกันภายใต้โค้งสปริงแบบเดียวกับแอปพลิเคชันระดับไฮเอนด์ (Apple-style smoothness):

```typescriptreact
// Page transitions & Dialog overlays
initial: { opacity: 0, y: 20, scale: 0.98 }
animate: { opacity: 1, y: 0, scale: 1 }
exit: { opacity: 0, y: -20, scale: 0.98 }

// Standard premium easing curve (Cohesive feeling)
ease: [0.22, 1, 0.36, 1]
```

*   **Spring Swiping Physics**: การปัดการ์ดหาคู่เดทใช้แรงต้านสะท้อนน้ำหนักนิ้วแบบธรรมชาติ
*   **Success Confetti Particle Glow**: เอฟเฟกต์แสงเรืองรองกระจายตัวเมื่อเกิดการอัปเกรดบัตรแพลตตินัมสำเร็จ
*   **Financial Calendar Transition**: แอนิเมชันการสไลด์เปิดดูปฏิทินการเงินรายวัน (Financial Calendar) แบบลื่นไหลด้วยสปริงฟิสิกส์

### 6. Mobile-First & Security Considerations

*   **Touch Targets**: ปุ่มกดและช่องกรอกข้อมูลมีความสูงมาตรฐาน 56px (สูงกว่าเกณฑ์สากล 44px) เพื่อหลีกเลี่ยงการกดพลาด
*   **One-Hand Operation Focus**: ตำแหน่งปุ่ม Call to Action, ปุ่ม Swipe Action (Like, Pass, Super) และปุ่มยืนยันผลลัพธ์ ถูกจัดวางให้อยู่ในโซนที่นิ้วโป้งเอื้อมถึงง่ายขณะถือโทรศัพท์ด้วยมือข้างเดียว
*   **PIN & Biometric Lock Page**: ระบบสกรีนล็อกแอปพลิเคชันก่อนเข้าสู่หน้าจอทางการเงินด้วยหน้าปัด PIN Code ขนาดใหญ่ กดง่าย พร้อมระบบสั่นสะเทือนสั้นเมื่อกดปุ่ม (Haptic Simulator)
*   **Offline-First Visual Indicator**: การแสดงผลไอคอนคลาวด์ขนาดเล็กที่มุมจอเมื่อบันทึกข้อมูลแบบออฟไลน์สำเร็จ และแสดงแอนิเมชันลูกศรหมุนวนเพื่อ Sync ข้อมูลเมื่อตรวจพบสัญญาณอินเทอร์เน็ตกลับมาเป็นปกติ
